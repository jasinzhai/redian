# 🔧 修复说明

## 问题
点击选择日期查看后，页面一直弹窗提醒，无法关闭。

## 原因
在 `loadDataForDate()` 函数中，当选择的历史日期数据文件不存在时：
1. 弹出 alert 提示"数据不存在"
2. 自动调用 `loadTodayData()` 加载今天的数据
3. 但 `loadTodayData()` 再次调用 `loadDataForDate(today)`
4. 如果今天的历史数据文件也不存在（因为默认数据是 hot_data.json），又会触发 alert
5. 形成循环弹窗

## 解决方案
1. 在 `loadDataForDate()` 函数中增加一个可选参数 `showAlert`，控制是否显示提示
2. 在 `loadTodayData()` 中调用 `loadDataForDate(today, false)`，不显示弹窗
3. 这样只会在用户手动选择日期且数据不存在时提示一次，不会循环弹窗

## 修改内容

### index.html
```javascript
// 修改前
async function loadDataForDate(date) {
    // ...
    alert(`${date} 的数据不存在，将显示今天的数据`);
    await loadTodayData();
}

async function loadTodayData() {
    const today = formatDate(new Date());
    document.getElementById('dateSelect').value = today;
    await loadDataForDate(today);  // 这里会再次弹窗
}

// 修改后
async function loadDataForDate(date, showAlert = true) {
    // ...
    if (showAlert) {
        alert(`${date} 的数据不存在，将显示今天的数据`);
    }
    await loadTodayData();
}

async function loadTodayData() {
    const today = formatDate(new Date());
    document.getElementById('dateSelect').value = today;
    await loadDataForDate(today, false);  // 不弹窗
}
```
